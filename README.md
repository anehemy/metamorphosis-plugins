# Metamorphosis Plugins

Claude Code plugin marketplace for [Metamorphosis Worldwide](https://metamorphosis-usa.com) products.

## Install

Inside Claude Code:

```
/plugin marketplace add anehemy/metamorphosis-plugins
/plugin install customer-clarity-free@metamorphosis-plugins
```

## Available plugins

| Plugin | What it does |
|--------|--------------|
| `customer-clarity-free` | Free edition of the Customer Clarity AI System: builds one customer avatar from your real call transcripts, in your customers' own words. See its [INSTALL.md](plugins/customer-clarity-free/INSTALL.md). |

More plugins will be added to this marketplace over time; `/plugin marketplace update metamorphosis-plugins` picks them up.
