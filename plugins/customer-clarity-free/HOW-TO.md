# How to Use This

Followable in under a minute. No git, no command line, no marketplace, nothing to install beyond this one file.

1. **Open Claude** (claude.ai).
2. **Add this skill.** Save the `SKILL.md` file somewhere you can find it, then add it as a Skill in Claude (look for "Add Skill" or an upload option in your Claude settings).
3. **Optional: connect Fathom or Microsoft 365** in Claude's connectors, so it can pull your recent call recordings automatically. Don't have either connected? No problem. You can paste a transcript from any source instead (Teams, Zoom, Otter, Fathom exports).
4. **Say: "Build my customer avatar."** Follow along. Claude will ask you a few quick questions about yourself first, so it knows whose voice to exclude, then walk you through the rest.

## What happens next

Claude will help you pull or paste a handful of real customer or sales calls, check with you on which ones actually look like customer conversations, and then build one clear picture of your real customer, in their own words, from what was actually said. If you don't have a filesystem available, everything shows up right in the chat, ready to copy.
